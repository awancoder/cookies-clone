Features

Everything you need to
transfer cookies

Simple, fast, and secure. Export cookies from one browser and import them to another in seconds.

📤

One-Click Export

Instantly grab all cookies from the active tab. Copy to clipboard or download as a JSON file.

📥

Easy Import

Paste cookies from clipboard or upload a JSON file. Preview before importing to ensure accuracy.

🌐

Auto-Detect Domain

Automatically detects the active domain and fetches all related cookies including subdomains.

👁️

Cookie Preview

View a detailed list of all cookies before exporting or importing. See name, domain, and expiry.

📊

Import Results

Get detailed statistics after import — see how many cookies were set successfully or failed.

🔓

Open Source

Fully open source and transparent. Inspect the code, contribute, or fork it for your own needs.

How It Works

Three simple steps to
clone your session

1
📤

Export Cookies

Open any website where you're logged in. Click the extension icon and copy all cookies to your clipboard or download them as JSON.

Browser A — Logged In ✅
Click "Copy to Clipboard"
2
📋

Transfer Data

The cookies are copied as JSON to your clipboard. You can also save them as a file and transfer to any device or browser.

{ "cookies": [ { "name": "session", "domain": ".facebook.com" } ... ] }
3
🚀

Import & Login

Open the same website in another browser. Paste the cookies, import them, refresh the page — and you're instantly logged in!

Browser B — Logged In ✅
🎉 Automatically Logged In!
